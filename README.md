# Bible Memorization App

Simple flashcard app for memorizing Bible verses. Select book/chapter/verse and create cards with reference on one side, text on other.

## Tech Stack

**Backend:**
- FastAPI
- Python 3.14+
- Loguru for logging

**Frontend:**
- Next.js 16
- React 19
- TypeScript
- DaisyUI + Tailwind CSS

## Setup

**Backend:**
```bash
# Install dependencies
uv sync

# Run dev server
uv run fastapi dev main.py
```

**Frontend:**
```bash
cd frontend/src

# Install dependencies
npm install

# Run dev server
npm run dev
```
